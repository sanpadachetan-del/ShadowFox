# 🚗 Car Selling Price Prediction using Machine Learning

## 📌 Project Overview
This project predicts the selling price of a used car using Machine Learning. The model analyzes various features such as car age, present price, kilometers driven, fuel type, seller type, transmission type, and owner details to estimate the selling price accurately.

## 📂 Dataset
The dataset contains information about used cars, including:
- Car Name
- Year
- Selling Price
- Present Price
- Kilometers Driven
- Fuel Type
- Seller Type
- Transmission
- Owner

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

## 🤖 Machine Learning Models
- Linear Regression
- Random Forest Regressor

## 📊 Results
- **Linear Regression Accuracy:** 84.65%
- **Random Forest Accuracy:** 96.45%

Random Forest achieved better performance and was selected as the final model.

## 📁 Project Files
- `Car_Price_Prediction.ipynb` – Complete notebook
- `car.csv` – Dataset
- `car_price_model.pkl` – Trained Machine Learning model
- `README.md` – Project documentation

## 🎯 Conclusion
This project demonstrates how Machine Learning can be used to predict the selling price of used cars. After comparing different models, Random Forest Regressor provided the best performance with an accuracy of **96.45%**.
